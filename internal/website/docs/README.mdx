# GraphQL EZ

`GraphQL EZ` is a Plugin-Based `GraphQL API Server` designed to be **Easy** and **Powerful**,
with great customization in mind.

This library is based on [**Envelop**](https://www.envelop.dev/),
a powerful Plugin-Based `GraphQL Execution orchestrator` that allows developers to build,
share and collaborate on GraphQL-related plugins while filling the missing pieces in GraphQL implementations.
You can read more about it [**here**](https://www.envelop.dev/docs).

With integrations for [**Fastify**](https://www.fastify.io/), [**Express**](https://expressjs.com/),
[**Hapi**](https://hapi.dev/), [**Koa**](https://koajs.com/), [**Next.js API Routes**](https://nextjs.org/docs/api-routes/introduction) and
[**Node.js HTTP**](https://nodejs.org/api/http.html):

## Features

Many of the features of GraphQL EZ are enabled through its plugin-based platform, allowing for example, one-line integrations of:

:ballot_box_with_check: First Class support for [TypeScript](https://www.typescriptlang.org/) using [Module Augmentation](https://www.typescriptlang.org/docs/handbook/declaration-merging.html#module-augmentation)

:ballot_box_with_check: Websockets support with both [graphql-ws](https://github.com/enisdenjo/graphql-ws) and [legacy subscriptions-transport-ws](https://github.com/apollographql/subscriptions-transport-ws) via <Link color="blue.500" href="/plugins/websockets" fontWeight="bold">Plugin</Link>

:ballot_box_with_check: [GraphQL Code Generator](https://www.graphql-code-generator.com/) via <Link color="blue.500" href="/plugins/codegen" fontWeight="bold">Plugin</Link>

:ballot_box_with_check: [GraphiQL IDE](https://github.com/graphql/graphiql) via <Link color="blue.500" href="/plugins/graphiql" fontWeight="bold">Plugin</Link>

:ballot_box_with_check: [Altair GraphQL Client](https://altair.sirmuel.design/) via <Link color="blue.500" href="/plugins/altair" fontWeight="bold">Plugin</Link>

:ballot_box_with_check: [GraphQL Scalars](https://www.graphql-scalars.dev/) via <Link color="blue.500" href="/plugins/scalars" fontWeight="bold">Plugin</Link>

:ballot_box_with_check: [GraphQL Voyager](https://apis.guru/graphql-voyager/) via <Link color="blue.500" href="/plugins/voyager" fontWeight="bold">Plugin</Link>

:ballot_box_with_check: And check the <Link color="blue.500" fontWeight="bold" href="/plugins">EZ Plugin Hub</Link> and <Link color="blue.500" fontWeight="bold" href="https://www.envelop.dev/plugins">Envelop Plugin Hub</Link> for many more available features
